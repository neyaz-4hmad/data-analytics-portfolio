## Skills Demonstrated

**Technical**
- `pandas` — groupby, agg, merge, pivot_table, pct_change
- `numpy` — where, select, NaN handling
- `matplotlib` — bar, line, scatter, subplots, twinx
- Statistics — IQR, correlation, std dev, rolling average
- Data cleaning — missing values, string ops, date arithmetic
- Feature engineering — derived columns, flags, scores
- Multi-table merges — 2 and 3 DataFrame joins

**Business**
- Translating numbers into board-level insights
- Recommendation → Evidence format
- Quantified recommendations with ₹ / $ values
- Severity calibration across HR, operations, and finance
- Storytelling across 10 distinct business domains
